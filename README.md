# Usage

```
Usage: ./bluefox [OPTION] [ARGUMENT]

Options:
        -h, Shows help menu for this command
        -l, Sets up listener for a specified port
        -c, Connects to a server based on a specified address and port

Format:
        ./bluefox -h
        ./bluefox -l <PORT>
        ./bluefox -c <IP> <PORT>

Examples:
        ./bluefox -l 1234
        ./bluefox -c 127.0.0.1 123
```
